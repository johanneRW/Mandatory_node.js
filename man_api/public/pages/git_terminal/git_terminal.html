Git
There are multiple Git commands, but the most used ones are:
  • git init (initializes a new Git repository in the current directory)
  • git clone 'github-repository-address' (clones a remote repository to the local machine)
  • git add -A (adds all new and modified files)
  • git add 'nameOfMyFilesIWantToCommit' (it is possible to add more than one at the time)
  • git commit -m "my commit descriptions" (commits changes to the repository)
  • git push (pushes changes to a remote repository)
  • git pull (pulls changes from a remote repository)
  • git status (shows the status of the modified files in the repository)
When you are working with your project, the ones you will use the most are:
  • git add
  • git commit -m "message"
  • git push

Terminal
When working with programming, it is useful to know some of the basic terminal commands.
Remember that you can use tab to autofinish the name when you are writing a file or directory name. If tab doesn't finish the name, try to write a couple more characters; there may be more than one possible file or directory.
The most used commands are as follows:
  • ls: List files. This will list files in your current location, handy if you forgot the name or are in doubt of the directory you want to access is located.
  • cd : Change directory. cd "the directory name" will change your directory to "the directory name".
  • cd .. : Will take you to the directory above your current directory.
The terminal is also used for running applications and installing packages.
